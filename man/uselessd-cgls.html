Name

uselessd-cgls — Show control group contents

Synopsis

uselessd-cgls [OPTIONS...]

Description¶

uselessd-cgls is a shell script, similar in intent to systemd-cgls, which can either display the cgroupfs hierarchy in a tree format, show the kernel threads which are tasks in the root hierarchy, or show the PIDs and comms of the services in the system slice.

Options¶

The following options are understood:

-h, --help¶

Prints a short help text and exits.

-t¶

Prints a listing of the cgroupfs hierarchy tree and breaks.

-k¶

Prints the PIDs and comms of the kernel threads in the root hierarchy and breaks.

-s¶

Prints the PIDs and comms of the services monitored in the system slice.

Exit status¶

On success 0 is returned, a non-zero failure code otherwise.
